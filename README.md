# minecraft-rs

Minecraft rewritten in rust

## FAQ

- **Does this project have relations with saturn client?** - No, i just wanted to test my rust skills
- **Will the generation also be written in rust**: Right now, no, later in a stable release, yes
- **What versions will you support?** currently my main focus is 1.8.9 because of it's popularity and simplicity.
- **Why?**: Rust is fast, i feel like i can create a project like this and probably be the first one to make such a project
